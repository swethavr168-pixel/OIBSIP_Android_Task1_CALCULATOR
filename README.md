# OIBSIP_Android_Task1_Calculator
Android Calculator app using Java &amp; XML that performs basic arithmetic operations – OIBSIP Internship Task.

## 📖 Overview
This is a simple Android Calculator application developed as part of the Oasis Infobyte Internship Program (OIBSIP) under the Android Development domain. The app performs basic arithmetic operations such as addition, subtraction, multiplication, and division with a clean and user-friendly interface.

---

## 🎯 Objective
The objective of this project is to build a functional calculator app using Android Studio while improving understanding of Android UI design, event handling, and Java programming logic.

---

## 🚀 Features
- Basic arithmetic operations (+, −, ×, ÷)
- Clean and responsive user interface
- Real-time calculation display
- Clear (C) button to reset input
- Error-free and smooth user experience

---

## 🛠️ Technologies Used
- Java
- XML
- Android Studio

---

## 📂 Project Structure
- `MainActivity.java` → Core logic for calculator operations  
- `activity_main.xml` → UI layout design  
- `res/drawable` → Button styles and icons  
- `res/values` → Colors, strings, themes  
- `AndroidManifest.xml` → App configuration file
